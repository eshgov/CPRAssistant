IMPROVED CPR ASSISTANT - EXECUTABLE VERSION
==========================================

QUICK START:
1. Double-click "ImprovedCPRAssistant.exe" to run
2. Grant camera permissions when prompted
3. Choose your mode and start practicing CPR!

FEATURES:
- Real-time CPR guidance and feedback
- Improved BPM calculation (last 4 beats)
- Better compression detection
- Cloud upload for model training
- Face blurring for privacy
- Visual metronome (flashing)
- Two modes: Walkthrough and Feedback

CONTROLS:
- Q: Quit the application
- N: Next step (in Walkthrough mode)
- S: Skip to compressions (in Walkthrough mode)
- U: Upload session data to cloud

SYSTEM REQUIREMENTS:
- Windows 10 or later
- Webcam/camera
- No additional software installation required

TROUBLESHOOTING:
- If camera doesn't work, try running as administrator
- Ensure good lighting for better pose detection
- Close other applications that might be using the camera

DISCLAIMER:
This application is for training purposes only. In real emergency situations, 
always call 911 and follow certified CPR guidelines.

For more detailed information, see README_EXECUTABLE.md in the main directory.
